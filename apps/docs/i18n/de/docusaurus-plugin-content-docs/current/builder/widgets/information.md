# Information

Dieser Abschnitt **enthält Widgets, die Informationen über Ihre Projektlayer** und -komponenten bereitstellen. 

## Layer

<div class="step">
  <div class="step-number">1</div>
  <div class="content"><b>Ziehen Sie</b> das <code>Layer</code> Widget per <b>Drag & Drop</b> auf ein Panel.</div>
</div>

<div class="step">
  <div class="step-number">2</div>
  <div class="content"><b>Klicken Sie</b> auf das <code>Layer</code> Widget, dann erscheinen die <b>Layer-Einstellungen</b> auf der rechten Seite. Hier können Sie einen <code>Titel</code> und eine <code>Beschreibung</code> hinzufügen.</div>
</div>

<div class="step">
  <div class="step-number">3</div>
  <div class="content">
    Durch Klicken auf das <img src={require('/img/icons/3dots.png').default} alt="More options" style={{ maxHeight: "20px", maxWidth: "20px", verticalAlign: "middle", marginRight: "4px" }}/> <code>Weitere Optionen</code> Symbol neben dem Layernamen können Benutzer auf folgende Optionen klicken kann: <code>Zum Layer zoomen</code>, <code>Datenquelleninfo</code>, und <code>Layer ausblenden/anzeigen</code>. Es gibt auch die Option zum <code>Herunterladen</code>, aber diese ist nicht für alle Betrachter verfügbar.
  </div>
</div>

<div style={{ display: 'flex', flexDirection: 'column', alignItems: 'center' }}>
  <img src={require('/img/builder/builder_layer.png').default} alt="recent datasets" style={{ maxHeight: "auto", maxWidth: "80%", objectFit: "cover"}}/>
</div>

::::info
Betrachter können die Sichtbarkeit der vorausgewählten Layer umschalten, aber sobald das Dashboard **neu geladen** wird, kehren die Layer zu ihren ursprünglichen Sichtbarkeitseinstellungen zurück.
::::
