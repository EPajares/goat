# Information

Dieser Abschnitt enthält die Elemente, die Ihr Dashboard ausmachen, wie Layer und andere Komponenten. Ziehen Sie einfach **per Drag & Drop** Widgets aus dem rechten Panel auf jedes Panel Ihres Dashboards. Passen Sie jedes Widget an, indem Sie darauf klicken, wodurch die **Widget-Einstellungen** im rechten Panel geöffnet werden.

### Layer

Nachdem Sie Layer zu Ihrem Projekt hinzugefügt haben, klicken Sie auf einen Layer, um auf seine **Layer-Einstellungen** im rechten Panel zuzugreifen. Hier können Sie den **Titel** und die **Beschreibung** des Layers anpassen.

Neben jedem **Layernamen** finden Sie die <code>Weitere Optionen</code>-Schaltfläche. Je nach Layer werden verschiedene Optionen angezeigt, wie: **Zum Layer zoomen**, **Herunterladen**, **Datenquelleninfo**, **Layer ausblenden/anzeigen**.

::::info
Betrachter können die Sichtbarkeit der vorausgewählten Layer umschalten, aber sobald das Dashboard **neu geladen** wird, kehren die Layer zu ihren ursprünglichen Sichtbarkeitseinstellungen zurück.
::::
