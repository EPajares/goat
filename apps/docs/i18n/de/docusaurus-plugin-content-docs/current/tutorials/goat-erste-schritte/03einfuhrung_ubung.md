# Einführung in die Übung

In diesem Tutorial erstellen wir eine umfassende Erreichbarkeitsanalyse, um zu bewerten, wie einfach Einwohner von Mannheim Supermärkte zu Fuß erreichen können. Diese Analyse hilft dabei, Gebiete mit guter oder schlechter Supermarkt-Erreichbarkeit zu identifizieren und Planungsentscheidungen zu unterstützen. Wir werden nur Daten verwenden, die im Katalog-Explorer von GOAT verfügbar sind.
