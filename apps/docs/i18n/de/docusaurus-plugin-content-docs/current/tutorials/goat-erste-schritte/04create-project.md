# ✏️ Projekt erstellen

Nach der Anmeldung gelangen Sie zur Workspace-Seite. Um ein neues Projekt zu erstellen:

1. **Klicken Sie auf die `+` Schaltfläche**, um ein neues Projekt zu erstellen
2. **Wählen Sie einen Ordnerspeicherort** und füllen Sie die Felder aus:
   - **Projektname:** "Supermarkt-Erreichbarkeit in Mannheim"
   - **Beschreibung:** "Analyse der Supermarkt-Erreichbarkeit zu Fuß in Mannheim zur Identifizierung unterversorgter Gebiete und Unterstützung stadtplanerischer Entscheidungen"
3. **Klicken Sie auf `Erstellen`**
