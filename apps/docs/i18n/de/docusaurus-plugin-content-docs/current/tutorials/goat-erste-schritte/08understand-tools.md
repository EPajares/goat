# Einzugsgebiete verstehen

Das **Einzugsgebiet** ist eines der mächtigsten Werkzeuge in GOAT für Erreichbarkeitsanalysen. Während GOAT viele andere Analyse-Tools wie Heatmaps, Puffer, räumliche Joins und Aggregationen bietet, konzentrieren wir uns hier auf das Einzugsgebiet-Tool, da es perfekt für unsere Supermarkt-Erreichbarkeitsanalyse geeignet ist.

## Was ist ein Einzugsgebiet?

Ein Einzugsgebiet zeigt genau, **wie weit Menschen von bestimmten Startpunkten innerhalb einer bestimmten Zeit oder Entfernung reisen können**. Es ist wie ein "Servicebereich" um einen Punkt herum - alle Orte, die innerhalb der definierten Grenzen erreichbar sind.

In unserem Fall: **Welche Gebiete in Mannheim sind innerhalb von 15 Minuten zu Fuß von einem Supermarkt erreichbar?**
