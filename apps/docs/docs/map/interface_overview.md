# Map Interface

**Opening a project will take you to the Map Interface**. Here, you can create, organize, and style layers, visualize spatial data, and harness all of GOAT's powerful spatial analysis capabilities.

<div style={{ display: 'flex', flexDirection: 'column', alignItems: 'center' }}>
  <img src={require('/img/map/interface/map_interface.png').default} alt="Map Interface Overview" style={{ maxHeight: "auto", maxWidth: "auto", objectFit: "cover"}}/>
</div> 

### Upper Bar

#### Left Side
- <code>Project menu</code>: **Click to return** to the home page, edit project metadata, delete the project, lock the map extent, report an issue, or read the privacy policy.
- <code>Project Name</code>: **Double-click to edit** the project name.
- <code>Last Saved</code>: **Shows the timestamp** of the last saved change.

#### Right Side
- <code>Data and Builder toggle switch</code>: **Switch between** the Map and Dashboard Builder interfaces.
- <code>Share Project button</code>: **Manage the sharing options**.
- <code>Documentation button</code>: **Access the GOAT documentation**.
- <code>Job Status</code>: **View the status** of ongoing and completed tasks.
- <code>User profile</code>: **Access account settings** and log out.

### Navigation Bar
The navigation bar on the left side of the map interface provides access to Layers and Legend.

- <img src={require('/img/map/interface/layer.png').default} alt="Layers" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Layers</code>: **Add and manage** your assets in the form of layers within your GOAT project.
- <img src={require('/img/map/interface/list.png').default} alt="Legend" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Legend</code>: **Visualize the legend** of the layers.

### Spatial Operations Bar
The spatial operations toolbar on the right side of the map interface provides access to various tools for managing and analyzing your layers:

- <img src={require('/img/map/interface/sliders.png').default} alt="Properties" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Properties</code>: **View layer info**, symbology, and visibility settings for the selected layer.

- <img src={require('/img/map/interface/filter.png').default} alt="Filter" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Filter</code>: **Filter your layer**. Read more in [Filter](../map/filter).

- <img src={require('/img/map/interface/palette.png').default} alt="Layer Design" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Layer Design</code>: **Design the style** of your layers. Read more in [Layer Styling](../category/layer-styling).

- <img src={require('/img/map/interface/toolbox.png').default} alt="Toolbox" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Toolbox</code>: Read more in [Toolbox](../category/toolbox).

- <img src={require('/img/map/interface/compass-drafting.png').default} alt="Scenarios" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Scenarios</code>: Read more in [Scenarios](../Scenarios).
