# Map Interface

**Opening a project will take you to the Map Interface**. Here, you can create, organize, and style layers, visualize spatial data, and harness all of GOAT's powerful spatial analysis capabilities.

<div style={{ display: 'flex', flexDirection: 'column', alignItems: 'center' }}>

#### Left Side
- <code>Project menu</code>: <strong>Click to return</strong> to the home page, edit project metadata, delete the project, lock the map extent, report an issue, or read the privacy policy.
- <code>Project Name</code>: <strong>Double-click to edit</strong> the project name.
- <code>Last Saved</code>: <strong>Shows the timestamp</strong> of the last saved change.

#### Right Side
- <code>Data/Builder toggle</code>: <strong>Switch between</strong> the Map and Dashboard Builder interfaces.
- <code>Share Project</code>: <strong>Manage the sharing options</strong>.
- <code>Documentation</code>: <strong>Access the GOAT documentation</strong>.
- <code>Job Status</code>: <strong>View the status</strong> of ongoing and completed tasks.
- <code>User profile</code>: <strong>Access account settings</strong> and log out.


### Navigation Bar
The navigation bar on the left side of the map interface provides access to Layers and Legend.

- <img src={require('/img/icons/layer.png').default} alt="Layers" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Layers</code>: <strong>Add and manage</strong> your assets in the form of layers within your GOAT project.
- <img src={require('/img/icons/list.png').default} alt="Legend" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Legend</code>: <strong>Visualize the legend</strong> of the layers.

### Spatial Operations Bar
The spatial operations toolbar on the right side of the map interface provides access to various tools for managing and analyzing your layers:

- <img src={require('/img/icons/options.png').default} alt="Properties" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Properties</code>: <strong>View layer info</strong>, symbology, and visibility settings for the selected layer.

- <img src={require('/img/icons/filter.png').default} alt="Filter" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Filter</code>: <strong>Filter your layer</strong>. Read more in [Filter](../map/filter).

- <img src={require('/img/icons/styling.png').default} alt="Layer Design" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Layer Design</code>: <strong>Design the style</strong> of your layers. Read more in [Layer Styling](../category/style).
  
- <img src={require('/img/icons/toolbox.png').default} alt="Toolbox" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Toolbox</code>: Read more in [Toolbox](../category/toolbox).

- <img src={require('/img/icons/compass-drafting.png').default} alt="Scenarios" style={{ maxHeight: "20px", maxWidth: "20px", objectFit: "cover"}}/> <code>Scenarios</code>: Read more in [Scenarios](../Scenarios).
