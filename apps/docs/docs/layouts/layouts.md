# Layouts Interface

The **Layouts** in GOAT allows you to create professional map reports and export them as PDF or PNG files. You can design custom layouts with maps, text, and other elements, and generate both single-page and multi-page reports.

<div style={{ display: 'flex', flexDirection: 'column', alignItems: 'center'}}>
  <img src={require('/img/layouts/layouts_interface.png').default} alt="Layouts Interface Overview in GOAT" style={{ maxHeight: "auto", maxWidth: "auto", objectFit: "cover"}}/>
</div>

## Settings Panel
The Settings Panel allows you to create a new layout, customize the overall layout properties, and print the layout. The customization includes page size, orientation, resolution, and export format. You can activate Snap to Guides and Rulers for a better alignment of elements while creating the layout.

### Create a new Layout

<div class="step">
  <div class="step-number">1</div>
  <div class="content">Click <code>+ add layout</code> on the top right to <b>create a new layout</b>.</div>
</div>

<div class="step">
  <div class="step-number">2</div>
  <div class="content">Select the available templates or the blank one.</div>
</div>

<div class="step">
  <div class="step-number">3</div>
  <div class="content">Configure the Layout Settings by selecting your preferred <code>page size</code> (A4, A3, Letter), <code>orientation</code> (Portrait or Landscape), <code>resolution</code>, and <code>export format</code> (PDF or PNG).</div>
</div>

<div class="step">
  <div class="step-number">4</div>
  <div class="content">Enable <code>Snap to Guides</code> and <code>Rulers</code> for precise element alignment.</div>
</div>

<div class="step">
  <div class="step-number">5</div>
  <div class="content">Now you are ready to start working on your Layout. Once is finished click <code>Print Layout</code> at the bottom to <b>export your layout</b> as PDF or PNG.</div>
</div>

## Layout Preview
The Layout Preview is the main area that displays the current layout you are working on. You can interact with the elements, move them around, and see how they will appear in the final report.

## Elements Panel
The Elements Panel contains all the tools you need to add and customize elements in your layout. You can add maps, text boxes, images, legends, scale bars, north arrows, and more.

<div class="step">
  <div class="step-number">1</div>
  <div class="content">Choose an element from the <code>Elements Panel</code> (Map, Text, Legend, Scale Bar, North Arrow, etc.).</div>
</div>

<div class="step">
  <div class="step-number">2</div>
  <div class="content"><b>Drag the element</b> onto the Layout Preview canvas to add it to your layout.</div>
</div>

<div class="step">
  <div class="step-number">3</div>
  <div class="content">Click on the element to <b>select and resize</b> it using the corner handles.</div>
</div>

<div class="step">
  <div class="step-number">4</div>
  <div class="content">With an element selected, use the <code>Properties Panel</code> on the right to <b>customize its appearance</b> and settings.</div>
</div>

## Layout Navigation

You can use the <code>zoom controls</code> on the bottom right to <b>zoom in or out</b> for detailed positioning. You can also click the <code>fit to screen</code> button to <b>see the entire layout</b> at once.
