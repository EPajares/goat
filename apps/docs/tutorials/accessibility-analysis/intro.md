# Introduction

Learn how to perform your first accessibility analysis in GOAT, from uploading data to creating beautiful catchment area maps.

## What You'll Learn

In this tutorial, you'll master:
- Creating a new project in GOAT
- Uploading and managing datasets
- Running catchment area analysis
- Styling and visualizing results
- Exporting your analysis

## Prerequisites

Before starting:
- A GOAT account ([sign up here](https://goat.plan4better.de))
- Basic understanding of accessibility concepts
- ~30 minutes of time

## Tutorial Overview

| Step | Topic                  | Duration |
| ---- | ---------------------- | -------- |
| 1    | Create a Project       | 5 min    |
| 2    | Upload Data            | 5 min    |
| 3    | Run Catchment Analysis | 10 min   |
| 4    | Style Your Results     | 5 min    |
| 5    | Export & Share         | 5 min    |
