# Style Layers

Now let's customize the appearance of both layers to create a clear map.

## Style the "Supermarkets" Point Layer

1. **Click on the "Supermarkets" layer**
2. **Disable `Custom Marker`** (markers will be converted to simple dots)
3. **Remove the "Color based on" field** by clicking the cross (×)
4. **Choose Orange as fill color**

## Style the "Mannheim" Boundary Layer

1. **Click on the "Mannheim" layer**
2. **Disable the fill color** in the right panel
3. **Enable the stroke color**
4. **Choose Black as border color** and set the **stroke width to 1**
