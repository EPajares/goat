# ✏️ Create Project

After logging in, you will arrive at the Workspace page. To create a new project:

1. **Click on the `+` button** to create a new project
2. **Select a folder location** and fill in the fields:
   - **Project Name:** "Supermarket Accessibility in Mannheim"
   - **Description:** "Analysis of walking accessibility to supermarkets in Mannheim to identify underserved areas and support urban planning decisions"
3. **Click `Create`**
