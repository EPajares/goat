# Exercise Introduction

In this tutorial, we will create a comprehensive accessibility analysis to assess how easily residents of Mannheim can reach supermarkets on foot. This analysis helps identify areas with good or poor supermarket accessibility and supports planning decisions. We will only use data that is available in GOAT's Catalog Explorer.
