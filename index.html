What is GOAT?

This is the experimental release of (GOAT).
GOAT or (Geo Open Accessibility Tool) is meant to be an open source, interactive, flexible and useful web-tool for accessibility planning. Its main data source is OpenStreetMap, however other data sources can be used as well. The backbone of the application is a PostgreSQL/PostGIS database..
